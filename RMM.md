# Richardson Maturity Model (RMM) #
  * Level 0
    * SOAP, XML RPC, POX
    * Single URI
  * Level 1
    * URI Tunnelling
    * Many URIs, Single verb
  * Level 2
    * Many URIs, many verbs
    * CRUD services (e.g. Amazon S3)
  * Level 3
    * Level 2 + Hypermedia
    * RESTful Services
